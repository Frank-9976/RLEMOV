FOR THE CASIO FX-9750GII

This tool converts a sequence images to a rlemov2 csv.
The file type is my invention, and its full name is "Run Length Encoded MOVie [version 2] for the casio fx-9750gii".
Program to parse rlemov2: does not exist.
Program to download parser / rlemov to calculator: here.
To load rlemov's, in CASIO FA-124, select File, CSV, Load from CSV, and then save the given csv (rlemov's are csv's) as a List File.
BONUS: To convert gifs to a sequence of images, I would use this cool website Calculator specs: 127 x 63 pixels, 2 colors, 20 frames.
This tool will autosize images to fit 127 x 63 and make images black and white.
